How to be an Author

What is an author?

An author is a person who writes books for a living, and then sells them for money through a publisher. They can write pretty much anything from How-To books to fiction novels. Anyone can be an author if their book gets officially published, so the possibilities are endless!

This website is dedicated to all the information you will need and can use in order to become an offical author!
